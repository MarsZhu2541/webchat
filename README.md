# Webchat

An online chat website based on:

Traditional Vue3 + Element plus  + Springboot + Redis + Keycloak + Docker

## Recommended IDE Setup

[Idea](https://www.jetbrains.com.cn/idea/)

## Project Setup

```sh
# change urls to localhost
mvn clean install
sh build.sh

cd keycloak
sh build.sh

cd frontend
sh build.sh
```
